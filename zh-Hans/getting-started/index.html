Getting Started

Installation#

Mapping Tools is an open source all-in-one collection of utilities for mapping in osu!.

It currently runs on only Windows operating systems.

Windows#

Mapping Tools can be downloaded from the downloads page.

Mac#

Unavailable

Mapping Tools is not yet supported yet for this platform.

Setup#

Once you've installed Mapping Tools, go to Options > Preferences to configure Mapping Tools.

The most important fields to set are the following:

  • osu! folder: path to the folder containing osu!.
  • Songs folder: path to the folder containing all your maps. It's typically directly inside the osu! folder.
  • osu! config user file: path to your osu! user configuration file. It's typically directly inside the osu! folder with the filename osu.[username].cfg.
note

Make sure these fields are correct, as Mapping Tools may not work properly otherwise.
